
# alarm clock

This project is a web application that utilizes the power of vanilla JavaScript to provide a seamless user experience. The main feature of the app is its abilty to set and delete alarm.

In addition to the set alarm feature, the app also includes the ability for users to view the upcoming alarms. Users can also delete the alarm if they want to.This web app also contains live timer.

Overall, this project demonstrates your skills JavaScript, and web development in general. The use of vanilla JavaScript allows for a lightweight and fast-performing application, without the need for any additional JavaScript frameworks. It's a great addition to your portfolio to show your ability to create a functional web application from scratch.


## Features

- set alarm
- delete alarm
- alarm ring
- responsive
